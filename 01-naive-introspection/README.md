# Naive Introspection

A GraphQL API has been deployed for a user management system. The developers left some debugging artifacts in the schema. Can you find the hidden flag?

## Difficulty

Easy

## How to Run

```bash
cd 01-naive-introspection
docker-compose -f docker-compose-dev.yml up --build
```

### Stopping the Challenge

```bash
docker-compose -f docker-compose-dev.yml down
```

## Accessing the Challenge

- **GraphQL Endpoint:** http://localhost:8001/graphql
- **GraphQL Playground:** http://localhost:8001/graphql
- **Health Check:** http://localhost:8001/health
